# Metodologia Acadêmica — ABNT & APA

## Visão Geral

Skill de referência para produção de trabalhos acadêmicos conforme as normas vigentes no Brasil (ABNT) e no padrão internacional (APA 7ª ed.). Cobre formatação, citações, referências e estrutura completa de TCC/monografia/artigo.

---

## 1. Formatação Geral (NBR 14724:2011)

| Elemento | Regra |
|---|---|
| Papel | A4 (21 × 29,7 cm) |
| Fonte | Times New Roman ou Arial, tamanho 12 |
| Margens | Superior e esquerda: 3 cm · Inferior e direita: 2 cm |
| Espaçamento corpo | 1,5 entre linhas |
| Espaçamento citação longa | Simples (1,0), recuo 4 cm à esquerda |
| Espaçamento referências | Simples, separadas por linha em branco |
| Parágrafo | Recuo de 1,25 cm na primeira linha |
| Numeração de páginas | Canto superior direito, a partir da 1ª folha textual |

## 2. Estrutura do Trabalho

### Pré-textual (sem numeração visível)
1. Capa (obrigatório)
2. Folha de rosto (obrigatório)
3. Folha de aprovação (obrigatório em TCC)
4. Dedicatória (opcional)
5. Agradecimentos (opcional)
6. Epígrafe (opcional)
7. Resumo em português + palavras-chave (obrigatório)
8. Abstract + keywords (obrigatório)
9. Lista de ilustrações / tabelas / abreviaturas (se houver)
10. Sumário (obrigatório)

### Textual (numeração visível)
1. Introdução
2. Desenvolvimento (capítulos)
3. Conclusão / Considerações finais

### Pós-textual (sem numeração visível no título)
1. Referências (obrigatório)
2. Glossário (opcional)
3. Apêndices (elaborados pelo autor)
4. Anexos (documentos externos)
5. Índice (opcional)

## 3. Citações (NBR 10520:2023)

### Indireta (paráfrase)
```
Segundo Fulano (2020), a democracia exige participação ativa.

A democracia exige participação ativa (FULANO, 2020).
```

### Direta curta (até 3 linhas) — no corpo do texto
```
Fulano (2020, p. 45) afirma que "a democracia exige participação ativa dos cidadãos".
```

### Direta longa (mais de 3 linhas) — em bloco recuado
```
    Texto da citação longa com recuo de 4 cm, fonte tamanho 10,
    espaçamento simples, sem aspas. (FULANO, 2020, p. 45-46).
```

### Dois autores
```
(FULANO; CICLANO, 2020)
```

### Três ou mais autores
```
(FULANO et al., 2020)
```

### Apud (citação de citação — usar com parcimônia)
```
(BELTRANO, 2015 apud FULANO, 2020, p. 30)
```
> Referência: citar apenas a obra consultada (FULANO, 2020).

### Supressão / interpolação
```
"[...] a democracia exige participação [grifo do autor]."
"[...] a democracia exige participação [grifo nosso]."
[interpolação entre colchetes quando o autor acrescenta palavras]
```

## 4. Referências (NBR 6023:2018)

### Livro
```
SOBRENOME, Nome. Título em negrito: subtítulo. Edição. Local: Editora, ano.
```
**Ex.:**
```
SILVA, João. Direito Constitucional Brasileiro. 3. ed. São Paulo: Atlas, 2022.
```

### Livro com dois autores
```
SILVA, João; SANTOS, Maria. Título. Local: Editora, ano.
```

### Capítulo de livro
```
SOBRENOME, Nome. Título do capítulo. In: SOBRENOME, Nome (org.). Título do livro. Local: Editora, ano. p. XX-XX.
```

### Artigo de periódico
```
SOBRENOME, Nome. Título do artigo. Nome do Periódico, Local, v. X, n. X, p. XX-XX, mês ano.
```

### Artigo online (DOI preferido)
```
SOBRENOME, Nome. Título do artigo. Nome do Periódico, v. X, n. X, e-XXXXX, ano. DOI: 10.XXXX/XXXX.
```

### Monografia / TCC / Dissertação / Tese
```
SOBRENOME, Nome. Título. ano. X f. Tipo (Grau) — Instituição, Local, ano.
```
**Ex.:**
```
ALVES, Camila. Violência doméstica e medidas protetivas. 2023. 98 f. Trabalho de Conclusão de Curso (Bacharel em Direito) — Universidade da Região de Joinville, Joinville, 2023.
```

### Legislação
```
BRASIL. Lei n. XXXXX, de DD de mês de AAAA. Dispõe sobre [...]. Diário Oficial da União, Brasília, DF, DD mês AAAA. Seção X, p. XX.
```

### Jurisprudência
```
BRASIL. Superior Tribunal de Justiça. Recurso Especial n. XXXXXXX/UF. Relator: Min. Nome. j. DD mês AAAA. DJe XX mês AAAA.
```

### Site / Documento online
```
SOBRENOME, Nome (se houver). Título da página. Nome do site, ano. Disponível em: https://url. Acesso em: DD mês AAAA.
```

## 5. Resumo (NBR 6028:2021)

- **Extensão:** 150–500 palavras (TCC/dissertação); 100–250 (artigo)
- **Forma:** parágrafo único, voz ativa, 3ª pessoa
- **Sem:** recuo de parágrafo, citações, abreviações não definidas
- **Estrutura sugerida:** objetivo → método → resultados → conclusão
- **Palavras-chave:** 3 a 6 termos, separados por ponto, em ordem alfabética (opcional)

```
Palavras-chave: Direito penal. Feminicídio. Medidas protetivas. Violência doméstica.
```

## 6. Sumário (NBR 6024:2012)

- Títulos em CAIXA ALTA para seções primárias
- Subtítulos em Caixa Alta e Baixa para secundárias
- Alinhamento com pontos de liderança até o número de página
- O sumário não se lista a si mesmo

## 7. APA 7ª Edição (referência rápida)

### Citação no texto
```
(Sobrenome, ano)          → narrativa: Sobrenome (ano)
(Sobrenome et al., ano)   → 3+ autores
(Sobrenome, ano, p. XX)   → citação direta
```

### Referência — livro
```
Sobrenome, N. (ano). Título em itálico. Editora. https://doi.org/...
```

### Referência — artigo
```
Sobrenome, N., & Sobrenome2, N. (ano). Título do artigo. Nome do Periódico, volume(número), páginas. https://doi.org/...
```

### Diferenças chave ABNT × APA

| Ponto | ABNT | APA |
|---|---|---|
| Autor | SOBRENOME, Nome | Sobrenome, N. |
| Data | após editora | após autor, entre parênteses |
| Título livro | **negrito** | *itálico* |
| Et al. | a partir de 3 | a partir de 3 |
| Citação direta | p. 45 | p. 45 |

---

## 8. Erros Comuns

| Erro | Correto |
|---|---|
| Ibid. / Op. cit. | Proibido na ABNT — repetir a referência completa ou abreviada |
| Aspas em citação longa | Sem aspas — o recuo já indica citação |
| "et alli" | Correto: **et al.** (com ponto) |
| Referências em ordem de aparição | ABNT: ordem alfabética pelo sobrenome |
| Data antes do título (ABNT) | Data fica após editora/local |
| Negrito no título de artigo | Somente livros e periódicos recebem negrito/itálico |
