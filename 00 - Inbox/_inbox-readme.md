# 00 — Inbox

Captura rápida de tudo que não tem contexto claro ainda. Novas ideias, anotações, rascunhos.

**Regra**: Tudo entra aqui primeiro. Quando o contexto fica claro, mova para a pasta correta:
- `01 - Projetos/<nome>/` se é sobre um projeto específico
- `02 - Áreas/<área>/` se é sobre uma área de vida (financeiro, marketing, aprendizado)
- `03 - Recursos/` se é um tutorial, documentação ou referência
- `04 - Arquivo/` se não é mais relevante (após arquivar)

**Tags sugeridas para Inbox**: `#ideia`, `#pendente`, `#rascunho`

---

Use este espaço livremente. O Claude vai sugerir a organização final quando ler suas notas.
