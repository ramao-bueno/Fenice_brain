❓ O que é Base de Conhecimento?

A base de conhecimento é o "cérebro" do seu bot

Contém documentos, artigos, FAQs e respostas que o bot utiliza para responder perguntas dos clientes. Quanto melhor a base, melhor o bot responde.

✅ O bot usa IA para encontrar a resposta mais relevante na sua base de conhecimento e responder de forma natural.

🚀 Método 1: Adicionar Documentos (PDF/TXT)

1
Acessar Knowledge Base

No Botpress Cloud, no menu lateral, clique em:

Knowledge → Knowledge Base → + Add Document
📍 Menu esquerdo > "Knowledge" > "Knowledge Base"
2
Fazer Upload de Arquivo

Formatos aceitos: PDF, TXT, DOCX, HTML

  • Clique em "Upload Document"
  • Selecione o arquivo do seu computador
  • O Botpress processa automaticamente
💡 Dica: Documente suas soluções, preços, processos em um PDF bem estruturado.
3
Nomear e Organizar

Após upload, o Botpress pedirá para nomear o documento:

Nome: "Soluções Fenice - Governança TI" Descrição: "Documentação completa sobre soluções" Categoria: "Produtos"
4
Revisar e Publicar

O Botpress mostrará um preview do documento processado. Revise e clique em "Confirm".

✅ O documento está agora disponível para o bot usar nas respostas.

✏️ Método 2: Adicionar Respostas Manuais (FAQ)

1
Criar Nova Entrada

Em Knowledge Base → + Add Document → Text Entry

Ou escolha "Add FAQ" para criar pergunta-resposta diretamente
2
Estrutura de FAQ

Exemplo para Fenice:

Campo Exemplo
Pergunta "Quais são as soluções Fenice?"
Resposta "Fenice oferece Governança de TI, O&M e Transformação Digital..."
Categoria "Produtos" ou "Serviços"
Tags "TI", "Consultoria", "Governança"
3
Salvar e Publicar

Clique em "Save" e a entrada fica imediatamente disponível.

⚙️ Método 3: Integrar Dados Externos (API)

Sincronizar com Sistemas Externos

O Botpress permite integrar dados de sistemas externos:

  • Confluence: Sincronizar wiki corporativo
  • SharePoint: Conectar documentos Microsoft
  • Notion: Usar base de dados Notion
  • HTTP API: Endpoints customizadas
💡 Para Fenice: Se tiver documentação em Notion ou Confluence, pode sincronizar automaticamente.
Como Configurar
Knowledge → Knowledge Base → Integrations → Escolha plataforma Autentique com suas credenciais (Google, Microsoft, etc) Selecione pastas/documentos para sincronizar

🔍 Testar a Base de Conhecimento

1
Acessar o Test Chat

No canto direito da tela, clique no ícone de chat para abrir o "Test Chat".

2
Fazer Perguntas

Teste o bot com perguntas relacionadas aos documentos:

"Quais são as soluções Fenice?" "Quanto custa uma consultoria?" "Como funciona o processo de implementação?"
3
Ajustar Base se Necessário

Se o bot não responder bem, adicione mais documentos ou refine as perguntas/respostas.

✅ Base de conhecimento é viva — atualize regularmente!

📊 Melhores Práticas

✅ Faça
  • Organize documentos por categorias claras
  • Use linguagem simples e natural
  • Inclua exemplos práticos
  • Atualize regularmente (pelo menos mensal)
  • Teste perguntas frequentes que recebe
  • Use termos que seus clientes usam
❌ Não Faça
  • Deixar documentos desorganizados
  • Usar jargão muito técnico sem explicação
  • Criar respostas muito longas (>500 palavras)
  • Ignorar feedback dos usuários
  • Deixar documentos desatualizados
  • Duplicar informações em múltiplos docs

📈 Monitorar Performance da KB

No Botpress, vá para Analytics → Knowledge Base para ver:

  • Documentos mais consultados
  • Perguntas que o bot não conseguiu responder
  • Taxa de relevância das respostas
  • Palavras-chave mais buscadas
💡 Use esses dados para identificar gaps na sua base e melhorar continuamente.
