# 02 — Áreas de Vida / Trabalho

Áreas permanentes que você cultiva continuamente.

## Organização Base

```
02 - Áreas/
├── Financeiro/       # Gestão financeira pessoal + empresa
├── Saúde/            # Bem-estar, saúde
├── Aprendizado/      # Cursos, estudos, desenvolvimento pessoal
├── Marketing/        # Conteúdo e estratégia de marketing da Próvision
│   ├── Blog/
│   ├── Social/
│   └── Campanha/
├── Cursos/           # Estrutura de cursos do Simulador
│   ├── 01-Viabilidade-Financeira/
│   ├── 02-Tributacao-Imobiliaria/
│   ├── 03-Financiamento-de-Obra/
│   ├── 04-Tutorial-Simulador/
│   ├── 05-Analise-de-Mercado/
│   └── assets/
├── Reports/          # Reports periódicos (semanais, mensais)
└── ...
```

Cada área pode ter suas próprias subpastas. Use livremente.

**Diferença com Projetos**: Projetos têm começo-meio-fim; Áreas são contínuas.

---

Novo conteúdo entra em `00 - Inbox/` e quando amadurece, migra pra Áreas conforme o contexto.
