# 📚 Fenice Brain — Sistema de Conhecimento Jurídico

Bem-vindo ao vault pessoal de conhecimento jurídico e legaltech.

## 📍 Estrutura do Vault (PARA)

```
Fenice brain/
├── 00 - Inbox/              Captura rápida (sem organização)
├── 01 - Projetos/           Projetos em progresso
├── 02 - Áreas/              Áreas permanentes de conhecimento
├── 03 - Recursos/           Material de referência
├── 04 - Arquivo/            Conteúdo fechado/histórico
├── 05 - Templates/          Templates prontos para usar
├── 06 - NotebookLM/         Recursos para análise com IA
│   ├── sources/            Documentos para processar
│   └── exports/            Resultados processados
├── _sistema/               Sistema e metadados
└── FENICE bRain/           Base jurídica (legal framework)
```

## 🎯 Como Usar

### Capturar Ideias
1. Crie notas no **00 - Inbox** sem se preocupar com organização
2. Use `[[ligações]]` para conectar com conteúdo existente

### Organizar Projetos
1. Crie projeto em **01 - Projetos** usando template
2. Use a estrutura PARA interna se necessário

### Construir Conhecimento
1. Adicione conteúdo em **02 - Áreas** para conhecimento permanente
2. Mantenha links para **03 - Recursos** e **04 - Arquivo**

### Integração com Notion
- Conteúdo pronto vai para **Sofia Próvision HQ** (equipe)
- Obsidian é seu espaço pessoal/bagunçado
- Notion é a versão limpa/acionável

## 📝 Arquivos de Sistema

- **_sistema/memory-log.md** — Decisões entre sessões
- **_sistema/tags-index.md** — Índice de tags
- **_sistema/notion-config.md** — Integração com Notion

## 🏷️ Tags Principais

Consulte `_sistema/tags-index.md` para lista completa.

Exemplos: #direito-civil #projeto-ativo #em-progresso #artigo

## 💡 Dicas

- Use o Inbox para capturar rápido
- Revise o Inbox 2x por semana
- Mantenha Áreas focadas em tópicos específicos
- Archive conteúdo concluído
