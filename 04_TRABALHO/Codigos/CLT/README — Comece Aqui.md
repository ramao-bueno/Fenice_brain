## 📝 Como Criar Novo Artigo

### Opção 1 — Copiar Template
1. Abra [[Templates/Template — Artigo CLT]]
2. Crie uma cópia
3. Renomeie: "Art. XXX — [Nome]"
4. Preenchaa as seções

### Opção 2 — Usar Obsidian Cli (Avançado)
```bash
# Se usar obsidian-cli
obsidian create --template "Template — Artigo CLT" "Art. 999 — Novo Artigo"
```
