# Árabe Clássico ‖ Alcorão — Hub

> بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ
> القُرْآن = "A Recitação". A língua é **constitutiva** do texto (إِعْجَاز) — a tradução é apenas
> تَرْجَمَة مَعَانِي القُرْآن (tradução dos significados).

## O objeto
- **Alcorão** (*Al-Qurʾān al-Karīm*) + **Sunnah** via **Hadith**.
- Língua: **Árabe Clássico / Alcorânico (Fusha turáthica)** — ≠ dialetos e ≠ MSA (*fuṣḥā al-ʿaṣr*).
- 28 letras · RTL · revelação em ~23 anos.

## Aparato de estudo
- Raiz triliteral (جَذْر) · صَرْف *ṣarf* + أَوْزَان · **Tajwīd** desde o início (o texto *é* recitação).
- Rasm ʿuthmânico + **قِرَاءَات** (Ḥafṣ × Warsh) → ver [[Nivel-3-Camada-Textual]].
- **عُلُوم القُرْآن** + Asbāb al-Nuzūl → ver [[Nivel-4-Ciencias-do-Sentido]].
- Hadith + **عِلْم الرِّجَال** + مَذَاهِب → ver [[Nivel-5-Fontes-e-Escolas]].
- Biblioteca: tradução **Helmi Nasr** (Liga Mundial Islâmica) / Samir El-Hayek + áudio de qāriʾ.
