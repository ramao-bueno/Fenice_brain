## 1. Visão Estratégica

**Fenice bRain** deixa de ser um vault pessoal e passa a ser uma **marca de conhecimento** — distribuindo o Sistema Acadêmico Jurídico Filosófico para o homem médio brasileiro (e depois global), em todos os canais onde ele está.

| Dimensão | Decisão |
|---|---|
| **Público** | Homem médio cultural — não especialista, não leigo absoluto. Quer entender a lei, não decorar artigos. |
| **Tom** | Didático, humano, acessível. Sem jargão desnecessário. Com profundidade quando importa. |
| **Filosofia** | Copiar os melhores → aprender com humildade → distribuir com ousadia |
| **Meta** | Plataforma mais bem remunerada da categoria no Brasil — e visível globalmente |

### Setup Inicial

**Idioma primário:** Português

### Formatos de Vídeo (misto)

| Formato | Duração | Exemplo de título |
|---|---|---|
| **Explainer** | 8–12 min | "O que é a Pirâmide de Kelsen? — A lei que organiza todas as leis" |
| **Lei comentada** | 5–8 min | "Art. 121 do CP: Homicídio — o que a lei diz de verdade" |
| **Caso prático** | 10–15 min | "O julgamento que mudou o Direito Penal brasileiro" |
| **Filosofia acessível** | 8–10 min | "Aristóteles tinha razão sobre a justiça?" |
| **Shorts** | até 60s | Definições, súmulas, princípios — 1 conceito por Short |

### Roteiro padrão (estrutura Khan + Kurzgesagt)

```
00:00 — Hook: pergunta que o homem médio já faz
00:30 — Por que isso importa para você (contexto)
01:30 — Conceito principal (explicação simples)
04:00 — Exemplo real / caso concreto
07:00 — Aprofundamento (para quem quer mais)
10:00 — Resumo em 3 pontos
11:00 — CTA: próximo vídeo / curso / vault
```

## 6. Apps Fenice bRain

### Stack recomendada (uma base → todos os destinos)

```
Flutter (Dart)
    ├── iOS (App Store)
    ├── Android (Google Play)
    ├── Windows Desktop
    └── Linux Desktop

Next.js PWA (já existe em fenice.ia.br)
    └── Web → instalável como app em qualquer dispositivo
```

### Por que Flutter?

- Uma base de código → 4 plataformas (iOS, Android, Windows, Linux)
- UI nativa em cada plataforma (não WebView)
- Google mantém ativamente
- Comunidade enorme

### Features do app Fenice bRain v1

| Feature | Descrição |
|---|---|
| **Busca de artigos** | Digita número do artigo → lei comentada |
| **Flashcards** | Repetição espaçada para OAB/concursos |
| **Vídeos** | YouTube integrado com notas do vault |
| **Filosofia do dia** | Citação diária de jurisconsulto ou filósofo |
| **Notificações** | Lembrete de estudo + novo vídeo |

## 7. Monetização — Modelo Completo

### Fase 1 — YouTube (0–6 meses)

| Fonte | Requisito | Estimativa BR |
|---|---|---|
| **YouTube Partner Program** | 1.000 subs + 4.000h assistidas | R$ 3–15/1.000 views |
| **Super Thanks / Membros** | Canal estabelecido | R$ 9,90–49,90/mês por membro |
| **Afiliados** | Cursos jurídicos parceiros | 20–40% por venda |

### Fase 2 — Produtos próprios (6–12 meses)

| Produto | Plataforma | Preço sugerido |
|---|---|---|
| **Curso "Direito para todos"** | Hotmart / Kiwify | R$ 97–197 |
| **Assinatura Vault Fenice** | Stripe via fenice.ia.br | R$ 29,90/mês |
| **OAB 60/30/10 — Pacote completo** | Hotmart | R$ 297–497 |
| **Mentoria em grupo** | Google Meet / Hotmart | R$ 197/mês |

### Fase 3 — Parceiros globais (12–24 meses)

| Parceiro | Tipo | Modelo |
|---|---|---|
| **Editoras jurídicas** (Saraiva, Foco, JusPodivm) | Patrocínio de vídeos | R$ 1.500–5.000/vídeo |
| **Cursinhos OAB** (Gran, Estratégia) | Co-marketing | % por conversão |
| **Softwares jurídicos** (Jurídico Certo, Astrea) | Afiliado/integração | % recorrente |
| **LegalTech global** | API Fenice | USD por chamada |

