## 🔗 Como Usar Fenice bRain

### 1. Consultar uma Lei
- Abra a pasta da área (ex: `02-Civil/`)
- Use Ctrl+K e busque o número do artigo (ex: "Art. 1º")
- Clique para abrir a nota completa

### 2. Estudar um Tópico
- Vá a uma pasta temática
- Leia o `INDEX.md` da área
- Siga os links internos para temas relacionados

### 3. Registrar uma Decisão Jurídica
- Crie uma nota em `02-Civil/Decisoes/` ou pasta equivalente
- Use o template: `_sistema/template-decisao-jurídica.md`
- Vincule aos artigos relevantes com `[[Art. XXX]]`

### 4. Acompanhar Atualizações
- Leia `_sistema/planalto-links.md` — links diretos para fontes
- Verifique `_sistema/changelog.md` — histórico de mudanças
- Procure por tag `#novo` nas notas recentes
