## Tags de Status

```
#vigente (Lei está em vigor)
#revogado (Lei foi revogada)
#alterado (Lei foi modificada)
#em-revisão (Projeto de lei pendente)
#2026 (Ano de alteração/vigência)
```

---

## 📌 Como Usar

**Cada nota deve ter:**
1. Uma tag de nível 1 (obrigatório)
2. Uma ou duas tags de nível 2 (obrigatório)
3. Uma ou duas tags temáticas (recomendado)
4. Uma tag de status (obrigatório)

**Exemplo de nota com tags corretas:**

```
---
tags: [civil, cc, capacidade, direitos-reais, vigente]
---
```
