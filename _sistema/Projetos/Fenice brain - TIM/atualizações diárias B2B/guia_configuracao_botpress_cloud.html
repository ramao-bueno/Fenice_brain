❓ O que é Botpress Cloud?

Plataforma de IA para Atendimento Automático

Botpress Cloud é uma plataforma enterprise de inteligência artificial especializada em automação de atendimento ao cliente e suporte. Oferece:

  • ✅ Chatbots inteligentes com IA generativa
  • ✅ Integração omnichannel (WhatsApp, telefone, chat web, email)
  • ✅ Escalação automática para agentes humanos
  • ✅ Análise de dados em tempo real
  • ✅ Integração com CRM, ERP, bases de conhecimento
  • ✅ Personalização sem necessidade de código
💡 Para Fenice Brain TIM: Ideal para automação de primeiro contato com clientes B2B, qualificação de leads e agendamento de reuniões consultivas.

⚙️ Passo 2: Configurar Fluxo de Conversa (Conversation Flow)

Estrutura do Bot para Fenice

Fluxo Recomendado:

Etapa 1: Saudação & Qualificação

"Olá! Seja bem-vindo à Fenice Tech. Você busca informações sobre soluções de Governança de TI, Transformação Digital ou Consultoria Estratégica?"

Ação: Coletar interesse do cliente

Etapa 2: Identificação da Necessidade

"Qual é sua função na empresa? (Selecione: CTO | Diretor TI | Gerente Operações | Outro)"

Ação: Segmentar cliente para perfil correto

Etapa 3: Coleta de Dados

"Qual é o tamanho da sua empresa? (1-100 | 100-500 | 500-1000 | 1000+)"

"Qual sua principal dor em TI ou Operações?"

Ação: Qualificar oportunidade

Etapa 4: Proposta de Valor

"Com base em suas necessidades, temos expertise em [solução específica]. Gostaria de conversar com um consultor especializado?"

Ação: Agendar reunião ou escalar

Etapa 5: Agendamento

Integração com calendário para agendar reunião diretamente com Ramão Bueno ou consultor disponível.

Ação: Criar evento no calendário

🔗 Passo 4: Integração com Sistema CRM/Salesforce

Conectar Dados de Clientes

Via Webhook (Recomendado):

1. Criar Webhook

No Botpress: Integrations → Webhooks → Create New

URL de Destino:

https://seu-crm.salesforce.com/api/v57.0/sobjects/Lead/
2. Mapear Campos

Configurar mapeamento de dados:

Campo Botpress Campo Salesforce
user.name FirstName / LastName
user.email Email
user.phone Phone
user.company Company
conversation.interest LeadSource
✅ Resultado: Cada conversa gera automaticamente um Lead em Salesforce para follow-up consultivo.

🎯 Passo 6: Configurar Escalação para Agentes Humanos

Transição Bot → Consultor

Configure regras para escalar para um humano quando necessário:

Trigger de Escalação 1: Complexidade

Se o cliente fizer pergunta que o bot não consegue responder → escalar

Trigger de Escalação 2: Intenção

Se cliente expressar interesse em agendar reunião → escalar para Ramão ou consultor disponível

Configurar Fila de Agentes

Em Integrations → Handoff Configuration:

  • Agente Primário: Ramão Bueno (ramao@fenice.tech)
