# SuperpowerFenice-05 — Filtro Epistemológico

**Skill real:** `.claude/skills/fenice-ia-02/skill.md`
**Invocar com:** `/fenice-ia-02`

## Função

Aplica uma lente filosófica específica como system-prompt de análise jurídica.
Transforma o agente em simulador de bancas acadêmicas, TCCs e pós-graduação.

## Lentes disponíveis

| Código | Filósofo |
|---|---|
| `POSITIVISMO_KELSEN` | Hans Kelsen |
| `UTILITARISMO_BENTHAM` | Bentham / Mill |
| `CONTRATUALISMO_HOBBES` | Hobbes / Locke / Rousseau |
| `HERMENEUTICA_GADAMER` | Gadamer |
| `CRITICA_FOUCAULT` | Foucault |
| `DIREITO_NATURAL_AQUINO` | Tomás de Aquino |

## Quando usar

- "Análise kelseniana de...", "visão foucaultiana de..."
- TCC ou artigo com recorte teórico específico
- Simulação de banca de pós-graduação
