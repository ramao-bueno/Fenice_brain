# SuperpowerFenice-07 — Exportador de Fichamento ABNT/APA

**Skill real:** `.claude/skills/fenice-ia-04/skill.md`
**Invocar com:** `/fenice-ia-04`

## Função

Para qualquer trecho jurídico selecionado (lei, súmula, doutrina), gera
instantaneamente: citação formatada ABNT, referência bibliográfica automática
e insight filosófico da IA como "Nota do Pesquisador".

## Fontes suportadas

| Fonte | Norma |
|---|---|
| Lei Federal | NBR 6023:2018 |
| Código | NBR 6023:2018 |
| Súmula STF/STJ | NBR 6023:2018 |
| Acórdão | NBR 6023:2018 |
| Doutrina | NBR 6023:2018 livro |

## Quando usar

- Escrita de TCC, artigo científico, monografia
- Fichamento de doutrina para concurso / OAB
- Exportação para Zotero/Mendeley via markdown
