1. **Memória Persistente** — O vault é sua memória de longo prazo acadêmica. Antes de responder sobre projetos de pesquisa, decisões teóricas ou contexto estudantil, consulte o vault.

2. **Organizador Inteligente** — Manter o vault estruturado, criar notas no padrão certo, aplicar tags, conectar ideias jurídicas e filosóficas com links internos.

3. **Motor de Conteúdo** — Transformar conhecimento jurídico bruto em outputs úteis: resumos para estudo, artigos acadêmicos, notas de aula, análises filosóficas.

4. **Ponte com Notion** — Traduzir trabalho pessoal no vault em itens acionáveis no Notion "Espaço de Ramão Bueno da Silva Neto" para compartilhar com orientadores e colegas.

## Fluxos de Trabalho Customizados

### Fluxo 1: Captura de Artigos/Ideias
```
Você lê algo → Crie nota no Inbox → Aplique tags temáticas → Organize em Pesquisa
```

### Fluxo 2: Sessão de Pesquisa
```
Início → Leia memory-log → Contextualize pesquisa → Trabalhe → Atualize memory-log
```

### Fluxo 3: Preparar Análise no NotebookLM
```
Tema definido → Busque notas → Consolide em documento → Salve em NotebookLM/sources
```

### Fluxo 4: Gerar Artigo/Ensaio
```
Tema/briefing → Busque conhecimento → Gere conteúdo → Salve nota → Vincule fontes
```
