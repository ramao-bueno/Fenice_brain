# Índice de Tags — Fenice Brain

Sistema centralizado de tags para organização e busca.

## Tags de Tipo
- `#nota` — Informação geral, anotação
- `#projeto` — Documentação de projeto
- `#recurso` — Tutorial, documentação, referência
- `#ideia` — Algo a explorar
- `#decisão` — Decisão registrada com racional
- `#jurídico` — Conteúdo jurídico

## Tags de Status
- `#rascunho` — Em elaboração
- `#em-progresso` — Trabalho ativo
- `#revisão` — Aguardando revisão
- `#pronto` — Pronto para uso/publicação
- `#arquivado` — Não é mais relevante
- `#privado` — Conteúdo privado, não publicar

## Tags de Conteúdo

- `#pendente` — Precisa de ação
- `#importante` — Alta prioridade

- `#dossiê-pessoal` — Informação pessoal/privada

---

**Como usar**: Copie a tag relevante ao criar/editar notas. As tags aparecem no frontmatter YAML.

Exemplo:
```markdown
---
tags: [projeto, direito-civil, em-progresso]
---
```
