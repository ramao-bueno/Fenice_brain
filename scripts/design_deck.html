A Solução

Fenice bRain em 3 passos

Do pedido ao prompt jurídico fundamentado em legislação real

1

Busque a Legislação

Envie sua dúvida jurídica em texto livre. O Full-Text Search em português recupera trechos relevantes com ranking semântico — com ou sem acentuação.

POST /buscar
2

Receba o Contexto

Os endpoints Premium preenchem templates de prompt — grounding jurídico, hermenêutica filosófica ou revisão de TCC — com os trechos legais recuperados.

POST /analisar
3

Injete no LLM

API REST

Endpoints disponíveis

GET /health Free

Status da API

Verifica operacionalidade e contagem de leis indexadas no Supabase. Sem autenticação.

POST /buscar Free

Busca FTS

Full-Text Search com ts_headline e ts_rank. Aceita query com ou sem acentuação. Retorna trechos com destaques em <b>.

POST /analisar Premium

Grounding Jurídico

Recupera legislação relevante e preenche o template grounding_juridico com contexto real para uso em LLMs.

POST /hermeneutica Premium

Hermenêutica Filosófica

Análise filosófica de textos jurídicos com contexto doutrinário recuperado automaticamente do vault Fenice.

POST /tcc Premium

Revisão de TCC

Agente revisor de rascunhos acadêmicos com suporte de legislação indexada pelo tema pesquisado.

GET /docs Free

Swagger UI

Interface interativa OpenAPI para explorar e testar todos os endpoints diretamente no navegador.
