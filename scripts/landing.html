API REST

Endpoints disponíveis

Nenhum endpoint chama LLMs diretamente. O campo prompt_preenchido é retornado pronto para injetar no modelo de sua escolha.

GET /health Free

Status da API

Verifica operacionalidade e retorna contagem de leis indexadas no banco Supabase.

POST /buscar Free

Busca FTS — sem autenticação

Full-Text Search com ts_headline e ts_rank. Funciona com e sem acentuação. Destaques em <b>.

POST /analisar Premium

Análise jurídica com grounding

Recupera legislação relevante e preenche o template grounding_juridico com contexto real para LLMs.

POST /hermeneutica Premium

Hermenêutica filosófica

Análise filosófica de textos jurídicos com contexto doutrinário recuperado do vault Fenice.

POST /tcc Premium

Revisão de TCC jurídico

Agente revisor de rascunhos acadêmicos com suporte de legislação indexada pelo tema pesquisado.

GET /docs Free

Swagger UI interativo

Interface visual para explorar e testar todos os endpoints diretamente no navegador, sem código.

Fluxo

Como funciona

Três passos do pedido ao prompt jurídico fundamentado em legislação real.

1

Busque a legislação

Envie sua dúvida jurídica para /buscar. O FTS em português recupera trechos relevantes com ranking semântico e destaques.

2

Receba o contexto

Os endpoints Premium preenchem templates de prompt — grounding, hermenêutica ou TCC — com trechos legais recuperados automaticamente.

3

Injete no LLM

O campo prompt_preenchido está pronto para enviar ao Claude, GPT-4, Gemini ou qualquer modelo. Nenhuma chamada é feita internamente.

Exemplo

Teste agora mesmo

Busca FTS sem autenticação — resposta em segundos com destaques no texto.

curl
# Busca Free — sem autenticação, sem cadastro curl -X POST https://fenice-justech.vercel.app/buscar \ -H "Content-Type: application/json" \ -d '{"query": "improbidade administrativa", "limite": 3}' # Resposta [ { "numero_ano": "Lei Federal 14230/2021", "tipo_ato": "Lei Federal", "ementa": "Altera a Lei nº 8.429 — improbidade administrativa", "trecho_relevante": "... <b>improbidade</b> <b>administrativa</b> ...", "relevancia": 1.0 } ]
